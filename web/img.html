Pikilap API Image

API链接

https://api-pikilap.pages.dev/image/img-json
https://api-pikilap.pages.dev/image/img


Img-Json

向此链接请求会得到如下内容
{
"filename": "u4HFCA0ZIWWgdvW.png",
"fileurl": "https://api-pikilap.pages.dev/image/img/u4HFCA0ZIWWgdvW.png",
"filesize": "64.22 KB",
"filewh": "700x700"
}
类型如下
filename 图片名
fileurl 图片URL
filesize 图片大小
filewh 图片长及宽
